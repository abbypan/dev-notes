apk-mitm
==========

安装 node.js
----------------

winget install OpenJS.NodeJS.LTS --accept-package-agreements --accept-source-agreements


安装 apktool
-----------------

https://apktool.org/


安装 uber-apk-signer
----------------------

https://github.com/patrickfav/uber-apk-signer


安装 apk-mitm
----------------

npm install -g apk-mitm


重打包apk
------------

apk-mitm some.apk
